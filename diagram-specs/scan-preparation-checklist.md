# Scan Preparation Checklist Diagram

## Purpose

A visual checklist/infographic summarizing the key preparation steps for a DEXA scan. Serves as a quick-reference companion to the detailed text content. Should be memorable and shareable — something a clinic could print as a handout.

## Visual Layout

**Format:** Vertical infographic with icon + text pairs, organized in a timeline/checklist format.

**Dimensions:** 500 x 600 viewBox

### Layout Zones

**Header (y: 0-60):**
- Title: "Preparing for Your DEXA Scan" in Poppins SemiBold, 20px
- Subtitle: "For the most accurate results" in Poppins Regular, 13px, muted text

**Checklist Items (y: 80-520):** Five items evenly spaced, each containing:

1. **Hydration** (y: 80-160)
   - Icon: Water droplet
   - Text: "Drink normally — don't over-hydrate or restrict fluids"
   - Color accent: Blue (#46C9F7)

2. **Food & Drink** (y: 170-250)
   - Icon: Fork and plate
   - Text: "Scan in a similar fed state each visit; avoid large meals 2hrs before"
   - Color accent: Salad green (#69D994)

3. **Exercise** (y: 260-340)
   - Icon: Dumbbell
   - Text: "Avoid intense exercise for 12-24 hours before scanning"
   - Color accent: Coral (#E87D7D)

4. **Clothing** (y: 350-430)
   - Icon: T-shirt
   - Text: "Lightweight, metal-free clothing (athletic wear is ideal)"
   - Color accent: #7C8D90 (muted)

5. **Timing** (y: 440-520)
   - Icon: Clock
   - Text: "Scan at a similar time of day for each visit"
   - Color accent: Amber (#F5C842)

**Footer (y: 540-580):**
- Key message: "Consistency matters more than perfection"
- Poppins Regular, 12px, italic, muted

### Each Item Structure

Left side: 40x40 icon circle with colored background
Center: Bold label text (14px) + description text (12px)
Right side: Checkbox icon (unfilled, for visual metaphor only)
